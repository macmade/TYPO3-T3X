1. IMPORTANT SECURITY NOTE
*******************************************************************************

A security problem has been fixed in version 0.1.3 of this extension.
All versions below version 0.1.3 have to be considered insecure, and should
not be used in any way.

If this file if present in your copy of the extension
(usually typo3conf/ext/ftpbrowser/README.txt), it means you are running a
secure version. Otherwise, please update immediately to a newer version:

http://typo3.org/extensions/repository/view/ftpbrowser/
